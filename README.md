# Interrupt-Driven Street Light Automation System

An STM32-based embedded system that automatically controls street lighting using ambient light intensity measured by a BH1750 digital light sensor.

## Features

- STM32 Microcontroller
- BH1750 Light Sensor
- I2C Communication
- Timer Interrupts
- UART Output
- Day/Dusk/Night Detection

## Hardware

- STM32 Nucleo Board
- BH1750
- LED
- UART Terminal
